## Overview

This repository contains a structured collection of **Computer Architecture laboratory experiments**. Each lab is organized in a separate folder and includes the required design files, testbench files, simulation outputs, screenshots, and documentation.

The main purpose of this repository is to document and demonstrate practical implementation of important Computer Architecture concepts using hardware description and simulation tools such as **VHDL**, **GHDL**, and **GTKWave**.

---

## Repository Objectives

The objectives of this repository are:

* To maintain all Computer Architecture lab reports in a clean and organized format
* To implement digital logic circuits using VHDL
* To simulate and verify circuit behavior using waveform analysis
* To understand fundamental computer architecture and digital design concepts through practical experiments
* To provide proper documentation for each completed lab

## Each Lab Folder Contains

Each lab folder includes the following required files:

| Required File          | Description                          |
| ---------------------- | ------------------------------------ |
| `README.md`            | Final lab report in Markdown format  |
| `.vhd` design file     | VHDL source code of the circuit      |

| `.vcd` simulation file | Generated waveform file              |
| Screenshot / Image     | Output waveform or simulation result |

## README Format for Each Lab

Every lab report contains the following sections:

1. Lab number and lab title
2. Objective
3. Theory
4. Implementation / Procedure
5. Output / Simulation result
6. Discussion
7. Conclusion

This format ensures that every lab submission is complete, readable, and properly documented.

## Current Completed Labs

### Lab 1: Introduction to VHDL Programming and Open-Source Simulation Environment

This lab introduces the basic workflow of VHDL programming using VS Code, GHDL, and GTKWave. It covers the structure of a VHDL design, simulation process, and waveform verification.

### Lab 2: VHDL Implementation and Verification of Basic Logic Gates

This lab focuses on implementing basic logic gates such as AND, OR, NOT, NAND, NOR, XOR, and XNOR using VHDL. The outputs are verified using simulation waveforms.

### Lab 3: Design and Simulation of Encoder and Decoder using VHDL

This lab demonstrates the design and simulation of a 4-to-2 Priority Encoder and a 2-to-4 Decoder using VHDL. The results are verified using GTKWave.

### Lab 4: VHDL Implementation of Multiplexer and Demultiplexer

This lab includes the design and simulation of a 4-to-1 Multiplexer and a 1-to-4 Demultiplexer using VHDL. The simulation verifies correct data selection and routing based on select lines.
